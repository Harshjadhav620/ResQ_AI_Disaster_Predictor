# ResQ_AI_Disaster_Predictor


## SetUp Guied 

### Install Requirement.txt file
``` pip install -r requirements.txt ```

### Run fastapi backend  ( use any command )
#### Command 1 
``` fastapi run dev ```

#### Command 2
``` uvicorn src.api.app:app --reload ```

#### Localhost backend url 
 
``` http://localhost:8000/predict ```
